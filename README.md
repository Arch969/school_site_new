# Сайт школы для Tilda

Проект разбит на исходники, чтобы сайт было проще редактировать и снова загружать в Tilda.

## Где редактировать

- `src/blocks/` - HTML-секции сайта.
- `src/css/` - стили по секциям.
- `src/js/` - скрипты по секциям.
- `dist/tilda.html` - готовый код для HTML-блока Tilda. Файл генерируется автоматически.
- `dist/preview.html` и `index.html` - локальный просмотр в браузере.
- `backups/` - резервная копия исходного монолитного файла.

## Команда сборки

```powershell
npm run build
```

Команда собирает `dist/tilda.html`, `dist/preview.html` и обновляет корневой `index.html`.

## Как загружать изменения в Tilda

1. Редактируйте файлы в `src/`.
2. Запустите `npm run build`.
3. Откройте `dist/tilda.html`.
4. Вставьте содержимое файла в HTML-блок Tilda.

Не редактируйте вручную `dist/tilda.html` и корневой `index.html`: при следующей сборке они будут перезаписаны.
